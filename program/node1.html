SITEBLAST commandline options

We will only discuss command line options that differ from the original BLASTZ distribution as in http://www.bx.psu.edu/miller_lab/.
Usually, the program is invoked by typing 
siteblast seq1 [seq2] options
where options can be one of the following and/or the standard BLASTZ options. Sequence data must be given in capital letters.
Subsections
  • Switch A, Printing option
  • Switch I, IUPAC seeds - key word tree search
  • Switch i, IUPAC seeds - trivial search
  • Switch D, Distance to IUPAC seeds.
  • Switch P, Seed search with position-weight-matrices
  • Switch pValue(required) and powerLimit(optional)
  • Switch power(required) and pValueLimit(optional)
